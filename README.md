
<h1>ALURA PLAY</h1>

<h2>En el curso de alura-play utilizamos HTML5, JAVASCRIPT y, principálmente, de CSS, Flexbox y Layouts responsivos en Alura Latam. Durante el mismo, construimos el proyecto,
